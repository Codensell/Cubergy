## GDD (MVP) — RU
## Cubergy

#### Жанр и платформа

Жанр: 3D-аркада / сбор ресурсов → трансформация → стрельба

Платформа: WebGL

Сессия: 2–5 минут

#### Игровая идея (fantasy)

Игрок начинает маленьким кубом, собирает энерго-кубы, “наращивает” себя до робота, затем до вооружённой формы, где игрок и есть оружие.

#### Основной цикл (Core loop)

Перемещение → сбор кубов → рост/трансформация → (повтор) → получение оружия → (позже) уничтожение врагов → победа.

#### Камера

3rd-person: камера следует сзади и чуть сверху под углом.

Плавное следование (smooth follow).

#### Управление

WASD — движение по плоскости.

Мышь — mouse-look (как в Counter-Strike), прицел фиксирован в центре экрана.

ЛКМ — заряд атаки (в форме с оружием).

#### Формы игрока (прогрессия)
Форма 0 — Hover Cube

Визуал: маленький куб (префаб).

Анимация: лёгкое “парение” вверх/вниз (bobbing) только у визуала (не у корня).

Способности: движение, сбор.

Порог: собрать 10 энерго-кубов → трансформация.

Форма 1 — Cube Walker

Визуал: “робот из кубиков” (префаб из примитивов).

Анимация: простая анимация ног/таза при ходьбе (без сложного рига в MVP).

Способности: движение, сбор.

Порог: собрать ещё 10 энерго-кубов → трансформация.

Форма 2 — Armed Walker

Визуал: как форма 1, но крупнее (scale ~ 1.5) + “ствол” (цилиндр/модуль) в центре.

Способности: движение, сбор, стрельба с зарядом.

Порог MVP: доступ к стрельбе (цели/враги добавляются позже).

#### Энерго-кубы (Collectibles)

Объект: куб с эмиссией/свечением, вращение/пульсация.

Подбор: триггер-коллайдер.

Спавн: фиксированные точки или поддержка N активных на сцене (например, 15).

#### Стрельба с зарядом (форма 2)

Удержание ЛКМ → заряд растёт от 0 до max за chargeTime.

При достижении max → авто-выстрел или выстрел по отпусканию (допускается оба варианта, но в MVP лучше: достиг max = выстрел, отпускание раньше = выстрел текущей силы).

Снаряд: простой projectile (сфера/капсула) + трейл.

Сила выстрела: влияет на скорость/урон/отдачу (минимум — скорость/дальность).

#### UI (минимум)

Счётчик: Energy: x/10 (для текущей формы).

Индикатор формы: Form: 0/1/2.

Слайдер заряда (показывать при удержании ЛКМ).

#### Уровень (блокинг)

Сцена из примитивов (платформы/стены/коридоры) в стиле sci-fi.

Границы: невидимые стенки, чтобы не выпадать.

Позже: заменить примитивы на меши из ассета, сохранив коллизии.

#### Враги и победа (после MVP-ядра)

Враги добавляются последним шагом.

Целевой финал джема: победа = уничтожить всех.

Stretch: враг на NavMesh, который тоже собирает кубы, трансформируется и начинает стрелять, если видит игрока.

#### Не делаем в MVP

Сложный ИИ, сложная физика, продвинутая анимация/риг, прокачка, много оружия.
